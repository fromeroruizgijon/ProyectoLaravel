Versión de laravel: 10
Función de las carpetas:
    App-> aquí se estructura el código principal, modelo y controladores

    routes-> aquí se definen las urls de la aplicación y que pasa cuando alguien entra en la ruta especificada

    Resource-> Contiene todo lo que el usuario ve, vistas blade, css, js, etc

    database-> todo lo relacionado con la base de datos

    storage-> archivos que genera la app

    public-> la pierta de entrada a la app, es la única carpeta que es accesible desde el esterior

    test-> pruebas automáticas

    vendor-> dependencias de laravel y librerías externas

    .env-> variables de entorno (credenciales y eso) no se sube al repo
